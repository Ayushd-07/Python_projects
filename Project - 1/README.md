# 🚀 Personal Data Collector

A simple yet beginner-friendly Python application that collects personal information from the user, displays it in a clean format, and performs basic calculations such as birth year estimation and height conversion.

---

## ✨ Overview

This project is designed to help beginners practice core Python concepts including:

- User input handling
- Variables and data types
- Type casting
- Formatted output
- Arithmetic operations
- Memory management basics using `id()`

---

## 🎯 Features

### 📝 User Information Collection
The program asks the user for:

- 👤 Name
- 🎂 Age
- 📏 Height (in feet)
- 🔢 Favourite Number

### 📊 Data Display
After collecting the information, the program:

- Displays all entered details in a structured format
- Shows the data type of each variable using `type()`
- Displays memory addresses using `id()`

### 🧮 Calculations
The application also:

- Estimates the user's birth year from their age
- Converts height from feet to centimeters
- Presents results with readable formatting
