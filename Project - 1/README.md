# 📌 Overview

The **Personal Data Collector** is a beginner-friendly Python application that gathers user information through console input and performs simple calculations.

The project demonstrates:

- User Input Handling
- Variables & Data Types
- Type Conversion
- Arithmetic Operations
- Formatted Output
- Built-in Functions

This project is ideal for Python beginners learning how data is collected, stored, processed, and displayed.

---

# 🎯 Problem Statement

### Objective

Build a console-based application that:

- Collects user information
- Stores data using appropriate data types
- Displays variable information
- Shows memory addresses
- Calculates birth year
- Converts height into centimeters
- Presents all results in a structured format

---

# ✨ Key Features

| Feature | Description |
|----------|------------|
| 👤 User Information Collection | Accepts name, age, height, and favourite number |
| 🔍 Type Identification | Displays variable data types |
| 🧠 Memory Address Display | Uses `id()` function |
| 🎂 Birth Year Calculator | Calculates estimated birth year |
| 📏 Height Converter | Converts feet into centimeters |
| 🖥️ Interactive CLI | Easy-to-use console interface |
| 🐍 Pure Python | No external libraries required |

# 🧠 Core Concepts

## User Input

```python
name = input("Enter your name: ")
```

## Type Conversion

```python
age = int(input())
height = float(input())
```

## Data Type Checking

```python
type(age)
```

## Memory Address Retrieval

```python
id(age)
```

## Birth Year Calculation

```python
birth_year = 2026 - age
```

## Height Conversion

```python
cm = height * 30.48
```
