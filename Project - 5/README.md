# рџ“Њ Overview

The **Employee Management System** is a Python console-based application developed using **Object-Oriented Programming (OOP)** concepts.

The project allows users to create different employee types, display their information, verify inheritance relationships, and demonstrates important OOP concepts like:

- Classes & Objects
- Inheritance
- Method Overriding
- Encapsulation
- Constructors
- Destructors
- Polymorphism
- `issubclass()` Function
